## 扱っている現象および本教材の説明

力の分解とは、1つの力ベクトルを互いに直交する2つの方向の成分に分けることです。力はベクトル量であり、方向と大きさをもちます。力を特定の方向に分解することで、運動の解析や斜面上での物体の動きを理解しやすくなります。

このシミュレーションでは以下の2つのモードで力の分解を体験できます。

### 水平・垂直分解モード（XY分解）

任意の力ベクトル $\vec{F}$ を、水平方向（x成分）と垂直方向（y成分）に分解します。

力 $\vec{F}$ の大きさを $F$、x軸正方向からの角度を $\theta$ とすると：

$$F_x = F \cos\theta$$
$$F_y = F \sin\theta$$

また、逆に成分から合力を求めることもできます：

$$F = \sqrt{F_x^2 + F_y^2}, \quad \theta = \arctan\!\left(\frac{F_y}{F_x}\right)$$

### 斜面分解モード

物体が斜面上に置かれているとき、重力 $mg$（鉛直下向き）を斜面に平行な成分と垂直な成分に分解します。斜面の傾き角を $\theta$ とすると：

- **斜面方向成分**（物体を滑らせようとする力）：$mg \sin\theta$
- **斜面垂直方向成分**（斜面を押す力・垂直抗力の反力）：$mg \cos\theta$

斜面の傾きが大きくなるほど $mg\sin\theta$ が大きくなり、物体が滑りやすくなることが視覚的に確認できます。

## 使用方法

### 水平・垂直分解モード

1. 左下の「水平・垂直分解」ボタンをクリックしてモードを選択します。
2. キャンバス中央の力ベクトル（黄色）の**先端をドラッグ**して力の方向と大きさを変えます。
3. 右上の「⚙ 設定」ボタンから、力の大きさや角度をスライダーで細かく調整できます。
4. 赤い矢印が水平成分 $F_x$、青い矢印が垂直成分 $F_y$ を表します。
5. 右下の数値パネルで $F$、$F_x$、$F_y$、$\theta$ の値を確認できます。

### 斜面分解モード

1. 左下の「斜面分解」ボタンをクリックしてモードを選択します。
2. 右上の「⚙ 設定」ボタンで質量と斜面角度を調整します。
3. 黄色の矢印が重力 $mg$、オレンジの矢印が斜面方向成分 $mg\sin\theta$、緑の矢印が垂直方向成分 $mg\cos\theta$ を表します。
4. 斜面角度を変化させて各成分がどのように変わるか観察しましょう。

## 観察のポイント

- $\theta = 0°$ のとき、$F_y = 0$（斜面では $mg\sin\theta = 0$）になり、力はすべてx方向（水平）に向くことを確認しましょう。
- $\theta = 90°$ のとき、$F_x = 0$（斜面では $mg\cos\theta = 0$）になり、力はすべてy方向（鉛直）に向くことを確認しましょう。
- $F_x^2 + F_y^2 = F^2$ の関係（三平方の定理）が常に成り立つことを確認しましょう。
- 斜面モードで角度を 0° から 90° に変化させたとき、$mg\sin\theta$ と $mg\cos\theta$ の大小関係が $45°$ で逆転することを観察しましょう。
