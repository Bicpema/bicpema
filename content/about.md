## Bicpema とは

**Bicpema**（ビックぺマ）は、「手元で動かしながら物理現象を観察する」をコンセプトにした、中学生・高校生向けの物理シミュレーション教材サイトです。

教科書に出てくるような物理現象を、ブラウザ上でインタラクティブに体験できます。数式だけでは伝わりにくい動きや変化を、自分の手で操作しながら確かめてみてください。

## 使い方ガイド

### 対象科目・単元

中学・高校の物理で学ぶ以下のカテゴリのシミュレーションを公開しています。

| カテゴリ | 対象となる単元の例                          |
| -------- | ------------------------------------------- |
| 力学     | 等速直線運動、自由落下、力の合成・分解 など |
| 波動     | 縦波・横波、波の反射、ドップラー効果 など   |
| 熱       | 熱の移動、比熱・熱容量、熱力学第一法則 など |
| 電磁気   | 直線電流の磁場、変圧器 など                 |
| 地学     | 半減期 など                                 |

### スマートフォン・PC どちらでも使える

Bicpema のシミュレーションはブラウザで動作するため、インストール不要で使えます。スマートフォン・タブレット・PC を問わず、授業中や自宅学習など、場所を選ばずに利用できます。

### 先生向けの活用例

- 授業の導入として、現象のイメージを視覚的に共有する
- 生徒が自分でパラメータを変えて「なぜそうなるか」を探究する活動に活用する
- テスト前の復習として、自分のペースで現象を確認する
